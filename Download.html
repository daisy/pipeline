For installation instructions see Installation.

Some packages do not include the command line tool. You can download it from https://github.com/daisy/pipeline-cli-go/releases.

Latest major version: 1.14.5

Includes the standalone desktop application, the server, and the command line interface. Versions available for Windows (an installer EXE), Mac OS (a DMG disk image containing a standard Mac OS application) and Linux (ZIP and DEB files). The DEB does not include the command line interface.

  • For Windows users (.exe file, 119 Mb)
  • For Mac OS X users (.dmg file, 135 Mb)
  • For Linux users (.zip file, 96 Mb)
  • For Debian/Ubuntu users (.deb file, 121 Mb)

  • For Windows users (.msi file, 53 Mb)
