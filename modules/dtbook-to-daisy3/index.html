DTBook to DAISY 3

Converts multiple DTBooks to DAISY 3 format.

Synopsis

Inputs

  • DTBook file(s)
    One or more 2005-3 DTBook files to be transformed. In the case of multiple files, the first one will be taken.
    Media-type:
    • application/x-dtbook+xml
    Sequence: yes
  • Text-To-Speech configuration file
    Configuration file for the Text-To-Speech.
    Configuration file for the Text-To-Speech.

    More details on the configuration file format.

    Sequence: no

Outputs

  • DAISY 3
    The resulting DAISY 3 publication.

Options

  • Publisher
    The agency responsible for making the Digital Talking Book available. If left blank, it will be retrieved from the DTBook meta-data.
    Possible values: Any <string>
    Default value:
  • Enable Text-To-Speech
    Whether to use a speech synthesizer to produce audio files.
    Possible values: true or false
    Default value: false
  • With text
    Includes DTBook in output, as opposed to audio only.
    Possible values: true or false
    Default value: true 
Usage: dp2 [GLOBAL_OPTIONS] dtbook-to-daisy3 [OPTIONS]

Options:
--source source One or more 2005-3 DTBook files to be transformed. In the case of multiple files, the first one will be taken.
--tts-config tts-config Configuration file for the Text-To-Speech.
--publisher [publisher] Publisher (default: )
--audio [audio] Enable Text-To-Speech (default: false)
--with-text [with-text] With text (default: true)
-o,--output [OUTPUT] Path where to store the results. This option is mandatory when the job is not executed in the background
-z,--zip Write the output to a zip file rather than to a folder
-n,--nicename [NICENAME] Set job's nice name
-r,--priority [PRIORITY] Set job's priority (high|medium|low)
-q,--quiet Do not print the job's messages
-p,--persistent Delete the job after it is executed
-b,--background Sends the job and exits
