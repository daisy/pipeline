CSS cascaders org.daisy.pipeline.css.CssCascader

  • media="embossed"

    CssCascader that supports media “embossed” and “print”. Only a very small subset of medium “print” is supported though, namely the properties color, font-style, font-weight, text-decoration.

XMLCalabash XProc steps (org.daisy.common.xproc.calabash.XProcStepProvider)

  • {http://www.daisy.org/ns/pipeline/braille-css}shift-id

    Move css:id attributes to inline boxes, see XProc documentation

  • {http://www.daisy.org/ns/pipeline/braille-css}shift-string-set

    Move ‘string-set’ declarations to inline boxes, see XProc documentation

  • {http://www.daisy.org/ns/pipeline/braille-css}render-table-by

    Render a table as a (nested) list, see XSLT documentation
