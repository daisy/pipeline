<p:declare-step version="1.0" name="main" type="px:daisy3-load"> <p:documentation> <p px:role="desc">Creates a fileset document based on a DAISY 3 package file.</p> <p>The fileset entries are ordered as the appear in the <code>manifest</code> element of the input OPF document, except for SMIL documents which are listed in the <code>spine</code> order. If more than one DTBook document is found in the fileset, SMIL documents are loaded in memory to sort the DTBook documents in the spine order in the resulting fileset. Otherwise, only the OPF itself is loaded in memory.</p> <p>Note: In the resulting fileset, the media type of SMIL documents will be <code>application/smil+xml</code> (as opposed to <code>application/smil</code> in DAISY 3) and the media type of the OPF document will be <code>application/oebps-package+xml</code> (as opposed to <code>text/xml</code> in DAISY 3).</p>
