9   /**
10   * Parses a comment in a specific format that is a subset of Doxygen:
11   *
12   * - Javadoc variant (C-style comment block starting with two *'s,
13   *   optionally with intermediate *'s at the beginning of each line)
14   * - optionally beginning with @var command:
15   *
16   *      @var <type> $<var-name>
17   *
18   * - optionally followed by @brief command
19   * - followed by a body of Markdown formatted text, with the first
20   *   line of the body being plain text (providing the short description)
21   *
22   * It is assumed that the following preprocessing has been done to the
23   * input before it is fed to the lexer:
24   *
25   * - check that first character is '*'
26   * - prepend new line character plus an amount of space characters
27       corresponding to the original column of the second '*' of the
28       opening '/**'
29   * - append new line character
30   */

32  public class DoxygenParser extends Parser {
