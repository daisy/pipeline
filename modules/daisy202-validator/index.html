DAISY 2.02 Validator

Validates a DAISY 2.02 fileset.

Synopsis

Inputs

  • NCC
    The NCC file in the DAISY 2.02 fileset
    Media-type:
    • application/xhtml+xml
    • text/html
    Sequence: no

Reports

  • Validation report

Options

  • Time Tolerance
    The duration of the audio files can deviate from the duration declared in the DAISY 2.02 fileset by this many milliseconds.
    Possible values: Any <integer>
    Default value: 500 
Usage: dp2 [GLOBAL_OPTIONS] daisy202-validator [OPTIONS]

Options:
--ncc ncc The NCC file in the DAISY 2.02 fileset
--timeToleranceMs [timeToleranceMs] Time Tolerance (default: 500)
-o,--output [OUTPUT] Path where to store the results. This option is mandatory when the job is not executed in the background
-z,--zip Write the output to a zip file rather than to a folder
-n,--nicename [NICENAME] Set job's nice name
-r,--priority [PRIORITY] Set job's priority (high|medium|low)
-q,--quiet Do not print the job's messages
-p,--persistent Delete the job after it is executed
-b,--background Sends the job and exits
