The index element represents a topical reference section in a book.

The entries in an index may be grouped into sections by topic or alphabetic letter.

Usage Example

<index>
    <section>
        <h>A.</h>
        <entry>ABERRANT groups, <ref>429</ref>.</entry>
        <entry>…</entry>
    </section>
</index>

Allowed parents

backmatter

Allowed children

This element must not contain text.

This element may contain the following children: aside, block, entry, h, hd, object, p, pagebreak, section and transition
