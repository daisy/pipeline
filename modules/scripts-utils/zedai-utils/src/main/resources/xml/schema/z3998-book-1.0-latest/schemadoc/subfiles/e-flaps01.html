The flaps element represents all content and images contained on the front and back jacket flaps.

If the content on the flaps is continued from the front to back flap, it should be merged and treated as unbroken.

Usage Example

<cover>
    …
    <flaps>
        <block role="photograph">
            <object src="author-pic.png" />
        </block>
        <block role="biographical-note">
            <p>Darwin was born in …</p>
            <p>After attending …</p>
        </block>
    </flaps>
</cover>

Allowed parents

cover

Allowed children

This element must not contain text.

This element may contain the following children: address, annotation, aside, block, ssml:break, caption, citation, code, description, d:description, xforms:group, hd, list, m:math, note, object, p, quote, xforms:repeat, sel:select, table, transition and verse
