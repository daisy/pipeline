DTBook to ODT

Transforms a DTBook (DAISY 3 XML) document into an ODT (OpenDocument Text).

ODT is the native file format of OpenOffice and LibreOffice but can be opened with Microsoft Word as well. The DTBook to ODT converter is designed to work with both.

Synopsis

Inputs

  • Template
    OpenOffice template file (.ott) that contains the style definitions.
    OpenOffice template file (.ott) that contains the style definitions.

    Defaults to default.ott.

    See Templating for more details.

    Media-type:
    • application/vnd.oasis.opendocument.text-template
    Sequence: no
  • DTBook
    The DTBook to transform.
    Media-type:
    • application/x-dtbook+xml
    Sequence: no

Outputs

  • ODT
    The resulting ODT file.

Options

  • Image resolution
    Resolution of images in DPI.
    Possible values: Any <integer>
    Default value: 600
  • Float page numbers
    Try to float page numbers to an appropriate place as opposed to exactly following print.
    Possible values: true or false
    Default value: true
  • Page numbers
    Whether to show page numbers or not.
    Possible values: true or false
    Default value: true
  • ASCIIMath handling
    How to render ASCIIMath-encoded formulas.
    Possible values: One of the following:
    ASCIIMATH
    MATHML
    Default value: ASCIIMATH
  • Images handling
    How to render images.
    Possible values: One of the following:
    EMBED
    LINK
    Default value: EMBED 
Usage: dp2 [GLOBAL_OPTIONS] dtbook-to-odt [OPTIONS]

Options:
--template template OpenOffice template file (.ott) that contains the style definitions.
--source source The DTBook to transform.
--image-dpi [image-dpi] Image resolution (default: 600)
--page-numbers-float [page-numbers-float] Float page numbers (default: true)
--page-numbers [page-numbers] Page numbers (default: true)
--asciimath [asciimath] ASCIIMath handling (default: ASCIIMATH)
--images [images] Images handling (default: EMBED)
-o,--output [OUTPUT] Path where to store the results. This option is mandatory when the job is not executed in the background
-z,--zip Write the output to a zip file rather than to a folder
-n,--nicename [NICENAME] Set job's nice name
-r,--priority [PRIORITY] Set job's priority (high|medium|low)
-q,--quiet Do not print the job's messages
-p,--persistent Delete the job after it is executed
-b,--background Sends the job and exits

Templating

The conversion works by converting DTBook elements to ODT elements with corresponding semantics (paragraphs, text spans, frames, images, formulas, lists, tables, notes, comments, etc.). However because ODT has a more limited set of elements, different DTBook elements are mapped to the same ODT element. In order to differentiate between DTBook elements they are given styles in ODT.

These styles can be edited afterwards in the ODT document, but more importantly they can be saved in a template (*.ott file) which can be provided to the script the next time (via the “Template” option).

The template that is used by default is default.ott.

What follows is a mapping from DTBook elements to ODT styles. There are different types of styles: paragraph styles, character styles, frame styles, list styles and section styles. (Page styles also exist but are not used by the converter.)
