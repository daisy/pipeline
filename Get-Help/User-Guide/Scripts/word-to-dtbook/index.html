Word to DTBook

Transforms a Microsoft Office Word (.docx) document into a DTBook XML file.

Synopsis

Inputs

  • Input Docx file
    The document you want to convert.
    Media-type:
    • application/vnd.openxmlformats-officedocument.wordprocessingml.document
    Sequence: no

Outputs

  • DTBook output
    Output folder of the conversion to DTBook XML

Options

  • Accept revisions
    If the document has revisions that are not accepted, consider them as accepted for the conversion.
    Possible values: Any <string>
    Default value: false()
  • Footnotes number prefix
    Add a prefix before the note's number if numbering is requested.
    Possible values: Any <string>
    Default value:
  • Pagination mode
    Define how page numbers are computed and inserted in the result
    Possible values: One of the following:
    custom
    Use numbers tagged with the style 'PageNumberDAISY' in the document
    automatic
    Use Word page breaks to compute and insert page numbers in content
    Default value: custom
  • Footnotes number suffix
    Add a text between the note's number and the note's content.
    Possible values: Any <string>
    Default value:
  • Translate character styles
    Possible values: Any <string>
    Default value: false()
  • Image resampling value
    Image resampling targeted resolution in dpi (dot-per-inch)
    Possible values: Any <string>
    Default value: 96
  • Document author
    Possible values: Any <string>
    Default value:
  • Image resizing
    Possible values: One of the following:
    original
    Keep image size
    resize
    Resize images
    resample
    Resample images
    Default value: original
  • Subject(s)
    Subject(s) to be added as dc:Subject metadata
    Possible values: Any <string>
    Default value:
  • Footnotes starting value
    If footnotes numbering is required, start the notes numbering process from this value
    Possible values: Any <string>
    Default value: 1
  • Document title
    Possible values: Any <string>
    Default value:
  • Document identifier
    Identifier to be added as dtb:uid metadata
    Possible values: Any <string>
    Default value:
  • Extract vector shapes (Experimental)
    Try to export inline shapes like diagrams or charts during conversion using Microsoft Word. If deactivated, those shapes will be replaced by their name and description in the result. Proceed with caution : Word must not be blocked by any dialog bound to it, or the process might crash or get stuck indefinitely
    Possible values: Any <string>
    Default value: false()
  • Document publisher
    Possible values: Any <string>
    Default value:
  • Footnotes position
    Footnotes position in content
    Possible values: One of the following:
    inline
    Inline note in content (after the paragraph containing its first reference)
    end
    Put notes at the end of a level defined in footnotes insertion level
    page
    Put the notes near the page break
    Default value: end
  • Footnotes insertion level
    Lowest level into which notes are inserted in content. 0 means the footnotes will be inserted as close as possible of its first call.
    Possible values: Any <string>
    Default value: 0
  • Footnotes numbering
    Customize footnotes numbering
    Possible values: One of the following:
    none
    Disable note numbering
    word
    Use original word numbering
    number
    Use custom numbering, starting from the footnotes start value
    Default value: none 
Usage: dp2 [GLOBAL_OPTIONS] word-to-dtbook [OPTIONS]

Options:
--source source The document you want to convert.
--accept-revisions [accept-revisions] Accept revisions (default: false())
--footnotes-numbering-prefix [footnotes-numbering-prefix] Footnotes number prefix (default: )
--pagination [pagination] Pagination mode (default: custom)
--footnotes-numbering-suffix [footnotes-numbering-suffix] Footnotes number suffix (default: )
--character-styles [character-styles] Translate character styles (default: false())
--dpi [dpi] Image resampling value (default: 96)
--creator [creator] Document author (default: )
--image-size [image-size] Image resizing (default: original)
--subject [subject] Subject(s) (default: )
--footnotes-start-value [footnotes-start-value] Footnotes starting value (default: 1)
--title [title] Document title (default: )
--uid [uid] Document identifier (default: )
--extract-shapes [extract-shapes] Extract vector shapes (Experimental) (default: false())
--publisher [publisher] Document publisher (default: )
--footnotes-position [footnotes-position] Footnotes position (default: end)
--footnotes-level [footnotes-level] Footnotes insertion level (default: 0)
--footnotes-numbering [footnotes-numbering] Footnotes numbering (default: none)
-o,--output [OUTPUT] Path where to store the results. This option is mandatory when the job is not executed in the background
-z,--zip Write the output to a zip file rather than to a folder
-n,--nicename [NICENAME] Set job's nice name
-r,--priority [PRIORITY] Set job's priority (high|medium|low)
-q,--quiet Do not print the job's messages
-p,--persistent Delete the job after it is executed
-b,--background Sends the job and exits
